# 项目简介

## 核心功能
- 用户注册/登录及个人信息管理
- 旅游景点信息的爬取、清洗与存储
- 景点数据分析与可视化展示
- 基于AI的个性化旅游路线推荐

## 适用场景
- 旅游规划与推荐
- 旅游数据分析
- 旅游产品开发

## 技术特点
- 采用Python和Django框架进行开发
- 使用MySQL数据库存储数据
- 前端采用Bootstrap框架实现响应式设计
- 集成百度地图API和ECharts图表库，提供数据可视化功能
- 基于DeepSeek大模型API的AI推荐模块
